# Fallout Terminal Sounds for VS Code

Transform your VS Code experience into the post-apocalyptic world of Fallout with authentic terminal sounds!

## Features

- **Authentic Fallout Terminal Sounds**: Uses the original sound files from Fallout games
- **Typing Sound Effects**: Random keystroke sounds play as you type
- **Power On/Off Sounds**: Terminal boot-up and shutdown sounds when activating/deactivating
- **Configurable Settings**: Adjust volume, sound delay, and enable/disable functionality
- **Perfect Companion**: Works great with Fallout Pip-Boy themes

## Usage

Once installed, the extension will automatically start playing Fallout terminal sounds when you type in VS Code. 

### Commands

- `Fallout Sounds: Enable` - Enable terminal sounds
- `Fallout Sounds: Disable` - Disable terminal sounds

### Settings

- `falloutSounds.enabled`: Enable or disable sounds (default: true)
- `falloutSounds.volume`: Volume level from 0.0 to 1.0 (default: 0.5)  
- `falloutSounds.keystrokeDelay`: Minimum delay between sounds in milliseconds (default: 50)

## Installation

1. Install the extension from the VS Code marketplace
2. Reload VS Code
3. Start typing and enjoy the authentic Fallout terminal experience!

## Recommended Theme

For the complete Fallout terminal experience, pair this extension with the **Fallout Pip-Boy Theme**.
